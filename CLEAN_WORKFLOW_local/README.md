# CLEAN_WORKFLOW_local

Ce dossier est une copie complète du workflow SoccerStats pour une utilisation locale.

Inclut tous les scripts, la base de données, la configuration et la documentation nécessaires pour le fonctionnement du workflow live et historique.

Copiez ce dossier sur votre ordinateur et installez les dépendances Python suivantes :

```
pip install requests beautifulsoup4
```

Vérifiez que le fichier `predictions.db` est bien présent pour l'accès aux données historiques et live.

Consultez les fichiers `README.md` et `METHODOLOGIE_COMPLETE_WORKFLOW.md` pour la documentation complète.
