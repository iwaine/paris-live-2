# CLEAN_WORKFLOW

Ce dossier contient uniquement les scripts et données validés pour le workflow de détection et monitoring live SoccerStats.

## Contenu
- Scripts principaux
- Fichiers de données (CSV, DB)

## Utilisation
1. Lancer la détection des matchs live :
   ```bash
   python3 soccerstats_live_selector.py --url https://www.soccerstats.com/
   ```
2. Utiliser les autres scripts selon le guide.

## Fichiers inclus
- soccerstats_live_selector.py
- recurrence_stats_export.csv
- predictions.db
